PlugMem：ICML 2026 · 即插即用的 LLM Agent 长期记忆系统
ICML 2026GitHub
论文开源SOTA
核心：提出插件式长期记忆模块，将 Agent 交互历史组织为紧凑、可复用的知识单元，以最小上下文开销实现跨任务记忆召回。
方法：任务无关的记忆图结构 + 可插拔集成方案，支持 Claude Code / OpenClaw 等编码 Agent 开箱即用，含可视化记忆图检查与交互式检索。
结果：在 LongMemEval 和 HotpotQA 上取得新 SOTA，记忆召回准确率显著优于原始交互历史检索方案。
复现：✅ 代码与权重开源，提供 Claude Code / OpenClaw 插件安装方式。

SemanticQA：ACL 2026 Oral · 语义推理基准揭示 LLM 深层理解盲区
ACL 2026 OralGitHub
论文开源Oral
核心：构建覆盖 19 类任务、四种短语类型的语义推理基准，系统评估 LLM 在习语、名词复合、轻动词等深层语义理解上的表现。
方法：设计四阶段测试框架，分离句法解析与语义推理，通过混淆配对实验定位模型真实理解能力边界。
关键发现：多数 LLM 在表面句法层面表现优秀，但语义推理准确率骤降；习语与名词复合结构是主要盲区，GPT-5 系列亦未突破。
复现：✅ 数据与评估脚本完整开源，含预训练模型微调方案。
